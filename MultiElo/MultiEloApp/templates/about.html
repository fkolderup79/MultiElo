Multiway Game Rating Explained

The Multiway Game Rating will give player's in your home game a rating. A player's rating will be adjusted after each game based on position and the significance of that position. This rating system is based on the standard ELO rating system where each player in a game will be matched up against each other. Uses of the Multiway Game Rating can be used for any multiway game such as Poker, Card Games, Board Games, Lawn Games etc
